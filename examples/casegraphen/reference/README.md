# CaseGraphen Reference Workflow

This directory contains the deterministic reference workflow for the
`casegraphen workflow reason` command.

Run it with:

```sh
cargo run -q -p casegraphen -- workflow reason --input examples/casegraphen/reference/workflow.graph.json --format json
```

The checked-in report at
`reports/workflow.reason.report.json` is generated by the same command with
`--output reports/workflow.reason.report.json`. It proves that the current
workflow graph produces a full reasoning report containing:

- readiness results;
- readiness obstructions;
- completion candidates;
- evidence boundary findings;
- projection results and projection views;
- correspondence records;
- evolution records.

The report is compared against live CLI output by the `casegraphen` integration
tests, so evaluator changes must update this reference intentionally.
